# Simpsons Fan Page

Aqui podemos conocer de cerca a nuestros personajes favoritos

## Para ejecutarlo debes seguir los siguientes pasos
1. `git clone https://github.com/ElvisGT/simpsons_fan_page.git`
2. `cd simpsons_fan_page`
3. `npm install`
4. `npm run dev`
